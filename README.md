# All things you need to handle Uyghur/Uighur language

## Build

```shell
poetry install
```

## Usage

```python
from uyghur.conversion import uey2uly

print(uey2uly('پلام، جهان'))
```

## Testing

To test python 3.8, install tox globally and run

```shell
tox
```
